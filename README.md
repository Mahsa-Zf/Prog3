# Time Series, Streaming, and Signal Data Processing Course

## Overview
This course focuses on **practical skills** for processing and analyzing **time series**, **streaming**, and **signal data** using popular data science tools such as:

- **Jupyter Notebooks**
- **NumPy**
- **Pandas**
- **Bokeh**


---

## Learning Outcomes

1. **Data Loading and Cleaning**  
   - Techniques for importing and preparing streaming data for analysis.

2. **Data Analysis**  
   - Exploratory Data Analysis (EDA) to uncover patterns and trends in time series and signal data.

3. **Data Visualization**  
   - Creating effective visualizations to communicate insights using tools like Bokeh.

4. **Maintainable Solutions**  
   - Writing reusable, maintainable code for real-world data processing pipelines.

---

## Key Outcomes

- Understand the complete **data processing pipeline**.
- Be able to handle diverse types of streaming data.
- Conduct exploratory data analysis with confidence.
- Create effective visualizations to present your findings.
- Build a strong foundation for future research and exploration in time series and signal analysis.


---

## Tools Used
You will work with the following tools throughout this course:

- [Jupyter Notebooks](https://jupyter.org/)
- [NumPy](https://numpy.org/)
- [Pandas](https://pandas.pydata.org/)
- [Bokeh](https://bokeh.org/)
